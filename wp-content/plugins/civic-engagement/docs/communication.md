# Communication Module

## Purpose

Grouped email communication system.

---

# Current Simplifications

* Email only.
* No SMS.
* No WhatsApp.
* No campaign automation.

---

# Admin Flow

Admin can:

* send grouped emails,
* select recipients by:

  * event
  * thread
  * rep
  * schedule
  * electoral area

---

# Email Features

* duplicate email filtering,
* basic personalization,
* reusable templates.

Example:

* Hello {name}
* Thanks for your recent participation...

---

# Important Rules

* No bulk marketing workflow.
* No analytics/open tracking.
* No unsubscribe engine.

---

# Future Possibilities

* SMS
* WhatsApp
* campaign automation
* reminders
* delivery tracking
